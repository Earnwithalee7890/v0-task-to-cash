# Contributing to TrueScore

First off, thank you for considering contributing to TrueScore! It's people like you that make TrueScore a great tool for the Farcaster community.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, create a new issue with a clear title and description.
* Include steps to reproduce the bug and details about your environment.

### Suggesting Enhancements
* Open a new issue with the tag "enhancement".
* Describe the feature you'd like to see and why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Follow the project's coding style (see `.editorconfig`).
3. Ensure your code builds without errors (`npm run build`).
4. Write clear, concise commit messages.

## Coding Standards
* We use **Prettier** for formatting.
* We use **ESLint** for linting.
* Indentation: 2 spaces.
